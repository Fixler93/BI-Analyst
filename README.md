# BI-Analyst

## Project Description
This project is focused on providing business intelligence solutions through data analysis, visualization, and reporting. The goal is to enable stakeholders to make informed decisions based on data-driven insights.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Fixler93/BI-Analyst.git
   ```
2. Navigate to the project directory:
   ```bash
   cd BI-Analyst
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
