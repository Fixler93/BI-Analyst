📊 משימה 5: חקירת דפוסי מלאי ע"י SQL

שלושה חומרים שונים מתנהגים בצורות שונות. חקור את הטבלאות "draws" (משיכות) ו-"arrivals" (הגעות) בעזרת הטרמינל. התאם לכל גרף את החומר הנכון (01, 02 או 03), והחלט האם לבצע הזמנה היום (Day 10) לפי זמן האספקה.

Free-Text SQL Terminal
📋 Dictionary
SELECT *, Worker_ID
FROM draws, arrivals
WHERE Qty > 50
AND Remarks = 'Urgent'
ORDER BY Date DESC

הרץ שאילתה בטרמינל או לחץ על הכפתורים למעלה כדי להציג נתונים.

* זמן אספקה (Lead Time): יום אחד
גרף ג' - מודל שיני מסור
* זמן אספקה (Lead Time): יומיים (2 Days)
