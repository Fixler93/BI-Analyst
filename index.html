Mission 3: Database Architecture

מסד הנתונים הקודם קרס לחלוטין! עליך לעצב מחדש את הארכיטקטורה מתוך מאגר העמודות שחילצנו בחירום.

שים לב: יש כאן הצעה לעמודות נצרכות, עליך להחליט מה לשים ומה לא. מחייבים אותך להוסיף עמודת "תיאור סטטוס" להזמנה – תחשוב האם נדרשת טבלה נוספת כדי שהמודל יעבוד בצורה תקינה.

הוראות עבודה:
1. גרור עמודות לטבלאות (הוסף טבלאות בעזרת הכפתור).
2. קליק על עמודה בטבלה מגדיר אותה כ-PK (מפתח ראשי), קליק שני מגדיר כ-Calc (מחושב). שלישי מוחק את הסימון.
3. הגדר רמת נרמול לכל טבלה.
4. הקישור (Foreign Key) מתבצע בעמודת תיאור הסטטוס (באמצעות לחצן 🔗).
